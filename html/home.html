We intend to contribute to the improvement of the competitiveness of the software industry through the StarLab project by developing the original technology for automatic repair of SW errors, and distributing it as an open source tool.

We research on fundamental technologies for software analysis, including

  • Static and dynamic software analysis
  • Automatic software analysis
  • Automatic software repair
  • Software synthesis
based on the understanding of programming languages.
