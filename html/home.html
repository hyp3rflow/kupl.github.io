Next generation software analysis technology for software safety and reliability

We research next generation software analysis technology to make safe and reliable software through the StarLab project. Specifically, based on our understanding of programming language, we develop automatic SW analysis technology, automatic SW patch technology, and automatic SW synthesis technology.

Development of high-performance automatic software analysis technology at the practical level

  • Develop software analysis technology for automatically detecting software bugs and vulnerabilities
  • Develop software repair technology for automatically fixing buggy or vulnerable software
  • Develop software synthesis technology for automatically writing programs on behalf of humans
