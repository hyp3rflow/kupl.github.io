Our Mission

Our goal is to develop highly-practical automated software repair technologies for ensuring safety and reliability of software. To this end, we research on fundamental techniques in the field of programming languages with applications to real-world software such as system software, AI software, and blokchain software.

  • Software Error Detection for precisely localizing errors and vulnerabilities in software.
  • Software Verification for rigorously validating the safety and reliability of repaired software.
  • Software Synthesis for efficiently generating patches that can repair software.

Current Research Topics

Software Error Detection

  • ADAPT: Testing Deep Neural Networks with Adaptive Neuron-Selection Strategy
    • A new white-box testing approach for deep neural networks. The key novelty is to combine white-box testing with an algorithm that adapts the neuron-selection strategy.
    [github] [ISSTA 20 paper]

Software Verification

  • NPEX: Null Pointer Exception X
    • NPEX is an open-source patch recommendation system for Java Null Pointer Exceptions (NPE). NPEX aims to help developers to maintain their projects NPE-free by automatically generating compilable patches for NPE.
    [github]
  • Graphick: Learning Graph-Based Heuristics for Pointer Analysis
    • Open source learning framework for generating graph-based heuristics for pointer analysis. We present a feature language to describe graph structures and an algorithm for learning analysis heuristics within the language.
    [github] [OOPSLA 20 paper]

Software Synthesis

  • SynthRL: Program Synthesis using Reinforcement Learning
    • Program synthesizer based on OGIS using learning-based agents trained with reinforcement learning. [github]
